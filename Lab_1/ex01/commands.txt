1) cp - эта команда копирует файл/папку из одного каталога в другой
	$ cp опции /путь/к/файлу/источнику /путь/к/файлу/назначения
2) touch - создает новый пустой файл

3) grep - поиск конкретного слова/предложения в файле

4) whoami - называет имя пользователя

5)lsb_release -a - показывает информацию о дистрибутиве(номер версии)

6)shutdown - выключает систему(-h задает промежуток времени now,+10)

7) ping - показывает, подключены ли вы к сети, и помогает определить качество связи

8) wget - позволяет скачивать файлы из интернета по указанному URL

9) useradd - создает нового пользователя.(введите useradd + имя пользователя)

10) uname - отображает сведения о системе(с флагом -а выдвет сведения об имени компьютера и версии ядра)
